    # Mathematics

    I am preparing my Master's thesis about the stable homotopy groups of the spheres. Following this line, I wrote a complete proof of the Freudenthal Suspension Theorem using the homological Serre Spectral Sequence. You can find the (unpublished) document here: Freudenthal Suspension Theorem.

    Moreover, here [\(\mathrm{mod} \; 2\)] (demo for mobile users) you can find an extension of the minimal free \(\mathcal{A}_2-\)resolution of \(\mathbb{F}_2\) given by Hatcher's spectral sequences book, computed with a python script made by me, except for some of SageMath's \(\mathrm{mod} \; 2\) linear algebra routines, addressing a related combinatorial problem. The script is a naive implementation of the process described in Hatcher's book, so it is far from being optimal because I am not considering more subtle aspects of the \(\mathrm{mod} \; 2\) Steenrod Algebra \( \mathcal{A}_2 \). Here you can find a plot that resembles the \( E_2 \) page of the Adams spectral sequence for the sphere (for \(p = 2, 3, 5, 7\)), computed using my program (minr.py).

    The analogous computations can be found in [\(\mathrm{mod} \; 3\)], [\(\mathrm{mod} \; 5\)] and [\(\mathrm{mod} \; 7\)]. Note the sparseness effect induced on the \(E_2-\)page of the Adams Spectral Sequence by taking increasing prime numbers. That suggests (correctly) that the most interesting case of computing the stable homotopy groups of the spheres corresponds to \(p = 2\).

    Currently, I am implementing Yoneda and Massey products in my program, obtaining the following example: the multiplicative structure in \(E_2\) related to a homotopy orbit (virtual) representation sphere spectrum.
